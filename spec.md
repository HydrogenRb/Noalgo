# 项目介绍
本项目设计了一个浏览器脚本，能帮助用户屏蔽小红书、知乎和B站的推荐算法的影响，从而保证人的专注，降低算法信息流的沉迷性，通过“训练”使得用户丢掉对于AI算法的依赖和爽感。

# 项目内容
本项目包含一个chrome的油猴脚本

# 项目使用方法
将油猴脚本安装到chrome上，当用户使用zhihu B站和小红书的时候，当用户沉迷推荐算法的时候自动黑屏，频率是10秒沉迷推荐算法，就会黑屏五秒并且显示倒计时
